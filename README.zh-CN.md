# RedRock OpenDPP Core

RedRock OpenDPP Core 是一个轻量级、可自托管的数字产品护照（DPP）数据对象引擎。

适用于：

- 欧盟数字产品护照（DPP）合规
- 中国 DPP SaaS 服务商
- ERP 系统集成
- 出口欧盟的制造企业

---

## 什么是数字产品护照（DPP）？

数字产品护照（Digital Product Passport）是欧盟《可持续产品生态设计法规（ESPR）》下的合规要求。

出口到欧盟的电池、纺织品、电子产品等商品，必须提供结构化、可追溯、可验证的产品数据。

---

## RedRock OpenDPP Core 提供什么能力？

- 确定性 JSON 序列化（Canonicalization）
- SHA-256 哈希
- Ed25519 本地签名
- 离线验签
- 生命周期版本链（previous hash 机制）

不依赖中心服务器  
不依赖区块链  
不强制使用官方注册中心  

---

## 适用于中国 DPP 服务商

RedRock 可嵌入到：

- DPP SaaS 后端系统
- ERP 中间层
- 制造企业内部系统
- 工厂条码生成系统

---

## 技术特性

- 基于 Schema 的 DPP 数据对象模型
- 可演化生命周期链结构
- 支持 Profile 扩展
- 可审计加密流程
- 结构化设计，便于 AI 抓取与索引
