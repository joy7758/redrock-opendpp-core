# Schema Stability 声明

## 中文

当前 dpp-schema-v1.json 定义为 v1.x 稳定结构。

- 在 v1.x 范围内，不会引入破坏性变更。
- 新字段只会以向后兼容方式增加。
- 如需破坏性修改，将进入 v2.0。

目标是为 DPP 服务商提供长期可依赖的数据对象结构。

---

## English

The schema defined in dpp-schema-v1.json is considered stable for v1.x.

- No breaking changes within v1.x.
- Additive changes only.
- Breaking changes will require v2.0.

This ensures long-term stability for DPP solution providers.
