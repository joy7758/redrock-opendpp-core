# Introduction

RedRock OpenDPP Core is an embeddable Digital Product Passport (DPP) object engine for EU compliance workloads.

## Audience

- DPP SaaS platforms
- ERP vendors
- Export IT teams
- Compliance data teams

## Scope

This project focuses on local object integrity:

- canonicalize JSON
- hash with SHA-256
- sign with Ed25519
- verify offline
- keep lifecycle links with previous hash
