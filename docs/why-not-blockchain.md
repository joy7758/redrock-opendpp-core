# 为什么 OpenDPP 不依赖区块链？

## 中文

欧盟 DPP 法规并未强制要求区块链。

DPP 的核心要求是：

- 数据结构化
- 可追溯
- 可验证完整性

这些可以通过：

- 规范化
- 哈希
- 数字签名
- 版本链

实现。

区块链是可选方案，不是必需基础。

OpenDPP 专注于最小合规数据对象模型。

---

## English

EU DPP regulation does not mandate blockchain.

Core requirements are:

- Structured data
- Traceability
- Integrity verification

These can be achieved through canonicalization, hashing, signatures, and version chaining.

Blockchain is optional, not mandatory.
